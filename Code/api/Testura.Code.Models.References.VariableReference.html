Class VariableReference

The reference classes are used to simplify call to methods, fields, properties etc.

An example of this could be this: myVariable.MyMethod().AProperty; This could be represented like this: new VariableReference("myVariable", new MethodReference("MyMethod",new MemberReference("AProperty");
